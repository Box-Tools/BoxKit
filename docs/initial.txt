---------------   ------------------
class/interface   Description
---------------   ------------------
block             - block interface

block_base        base class for a block
                  - block size, block bound
                  - block data list
                  - convert var names to dictionary keys
                  - handles 2D/3D implementation

block_input       - class for input block

block_output      - class for output block

block_general     - inherits from block_input and block_output

grid              - grid interface

grid_base         base class for a grid
                  - do not inherit from blocks but create them
                  - a list of input blocks from i/o
                  - [num blocks, block data, block size, block bound]
                  - handles 2D/3D implementation
                  - optional - 'input' (default), 'output', 'general'
                  - store neighbor information

grid_spatial      - spatial grid (default)

grid_general      - spatio-temporal grid
